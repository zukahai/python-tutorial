# Xử lí bit

And bit trong python

```python

# and

a = 60            # 60 = 0011 1100
b = 13            # 13 = 0000 1101
c = a & b         # 12 = 0000 1100

print("Line 1 - Value of c is ", c)

```

# Xử lí bit

Or bit trong python

```python

# or

a = 60            # 60 = 0011 1100
b = 13            # 13 = 0000 1101
c = a | b         # 61 = 0011 1101

print("Line 1 - Value of c is ", c)

```

# xor
